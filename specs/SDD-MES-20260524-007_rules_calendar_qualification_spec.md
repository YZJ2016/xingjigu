## 2. 背景与目标

### 2.1 当前系统现状

- 现有基础资料包含产品、物料、BOM、工艺路线、工序工位、产线车间、客户供应商。
- 系统设置包含人员账号、角色权限、审批设置、单据同步、消息提醒、操作记录、数据备份。
- 真实 MES 还需要编码规则、不良/停机/报废/返工/报警原因代码、开工准入规则、抽检规则、班次日历、人员资质等生产级底座。

### 2.2 业务目标

- 补齐生产执行依赖的规则与代码体系，支撑开工准入、投料防错、质量检验、异常归因、设备停机和追溯。
- 将“人员账号”与“人员资质”区分开：账号负责登录身份，资质负责生产准入和签核资格。
- 将“产线车间”与“班次日历/资源日历”区分开：车间是组织资源，日历是排程和产能约束。

### 2.3 非目标

- 不实现通用规则引擎或技术状态机页面。
- 菜单不使用“状态机”“死信”“幂等”等技术词。
- 不接入真实 HR、PLM、QMS 或 APS。

## 3. 菜单建议

### 3.1 新增或调整入口

| 所属一级菜单 | 建议子菜单 | 业务定位 |
| --- | --- | --- |
| 基础资料 | 规则与代码 | 编码规则、不良代码、停机原因、报废/返工原因、报警等级、抽检规则、准入规则 |
| 基础资料 | 班次日历 | 班次窗口、资源日历、例外停线、保养窗口、瓶颈资源日历 |
| 系统设置 | 人员资质 | 岗位技能、质量签核资格、设备点检资格、特殊工序资格 |

### 3.2 可替代方案

- 如不新增菜单，可将 `产线车间` 改为 `产线日历`，但必须保留车间/产线组织模型。
- 如不新增 `人员资质`，可将 `人员账号` 页面拆出资质 Tab，但菜单文案要体现生产准入。

## 4. 业务闭环定义

### 4.1 正向主流程

```text
基础规则发布
  -> 订单评审 / 排程 / 派工 / 开工准入引用
  -> 现场模拟回执校验
  -> 异常拦截、质量签核、报工复核
  -> 追溯与报表引用同一规则版本
```

### 4.2 规则对象

| 对象 | 上游来源 | 下游引用 | 影响场景 |
| --- | --- | --- | --- |
| 编码规则 | MES 基础资料 / PLM 引用 | 批次、SN、箱码、托盘码、标签 | 条码生成、追溯 |
| 不良代码 | QMS / MES 质量基础 | 不良记录、返工评审、良率分析 | NCR、MRB、报表 |
| 停机原因代码 | 设备 / 生产管理 | 停机归因、停机损失、OEE | 设备维修、OEE |
| 报废/返工原因 | 质量 / 生产管理 | 不良记录、返工评审、物料损耗 | MRB、成本损耗 |
| 报警等级 | 设备 / 工艺 | 报警记录、异常处理、停线申请 | SLA、升级 |
| 抽检规则 | QMS / MES 质量 | 首件、巡检、过程检验、FQC | 检验任务生成 |
| 开工准入规则 | MES 工艺 / 质量 / 设备 / 物料 | 开工检查、扫码开工 | 人机料法环测准入 |
| 班次日历 | 生产管理 / 设备保养 | 排程、产能负荷、电子看板 | APS、加班、停线 |
| 人员资质 | HR / QMS / 设备管理 | 员工登录、点检、质量放行 | 权限、准入、签核 |

## 6. 交互规格

| 动作 | 前置条件 | 校验规则 | 成功结果 | 失败结果 |
| --- | --- | --- | --- | --- |
| 发布规则 | 草稿或待审批 | 必填范围、版本、责任人、审批意见 | 状态变为已发布，写入影响范围 | Toast 提示缺失项 |
| 停用规则 | 无执行中关键引用或允许替代版本 | 必须填写原因 | 状态变为已停用，影响下游准入 | 阻止停用并展示引用 |
| 资质复核 | 人员在职且资质未过期 | 岗位、工序、有效期必填 | 更新资质状态，影响登录准入 | 生成资质异常 |
| 日历例外维护 | 车间或产线已启用 | 起止时间、原因、影响资源必填 | 排程和产能负荷读取例外 | 阻止与已发布计划冲突 |

## 7. 验收标准

- 菜单新增或页面拆分后，能明确找到规则代码、班次日历、人员资质能力。
- 开工检查、扫码开工、质量放行、设备点检页面能展示至少一条规则或资质引用。
- 停机归因、不良记录、报警记录页面能展示原因代码或等级来源。
- 发布、停用、复核动作写入履历，包含责任人、时间戳、版本、影响范围。
- 不出现“状态机中心”等技术菜单名。

## 8. 实施顺序

1. 建立规则与代码种子数据。
2. 新增或扩展基础资料页面入口。
3. 补班次日历和人员资质展示。
4. 将规则引用接入开工、质量、设备、异常页面。
5. 验证规则变更对下游准入和追溯的展示影响。
