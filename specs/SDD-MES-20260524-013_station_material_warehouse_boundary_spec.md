## 2. 业务边界

- 工位动作来自工位终端、扫码枪、HMI、PDA、工牌/NFC 或模拟回执，后台只做监控、校验、拦截、补录、复核和追溯。
- 物料与仓储需兼容有 WMS 和无 WMS 两种模式。
- 无 WMS 时，MES 可以登记库存出入库和线边库记录，但必须具备库位、批次、数量、责任人、复核和追溯。

## 3. 工位作业整改

| 页面 | 整改要求 |
| --- | --- |
| 工位身份确认 | 菜单可改为“工位身份回执”；体现工牌/NFC/设备账号模拟回执 |
| 扫码开工 | 所有动作必须使用“模拟扫码”“模拟回执”“准入拦截”表达 |
| 工艺指导 | 展示 SOP 来源、版本、生效范围、终端签收和未签收拦截 |
| 投料确认 | 保留模拟标签、模拟称重、模拟投料回执；后台只做校验和拦截 |
| 过程记录 | PLC、HMI、测试台、人工录入均必须标明模拟回传和采集来源 |
| 工序报工 | 有 ERP 时生成回传消息并接收模拟 ERP 回执；无 ERP 时写入 MES 报工实绩和工单成本口径 |
| 交接班 | 交班/接班为模拟工牌/NFC 或责任人确认回执 |

## 4. 物料与线边库整改

| 页面 | 整改要求 |
| --- | --- |
| 用料需求 | 由排程、派工、BOM、库存和替代料规则生成，支持 MES 手工 BOM |
| 领料申请 | 有 WMS 接收 WMS 回执；无 WMS 登记 MES 领料出库记录 |
| 配送进度 | 接收模拟 PDA/AGV/人工签收回执 |
| 线边库存 | 有 WMS 生成补料请求；无 WMS 登记线边库补料和批次状态 |
| 投料记录 | 接收工位投料模拟回执，写入批次谱系 |
| 余料退回 | 有 WMS 接收回执；无 WMS 登记 MES 退料入库记录 |
| 缺料预警 | 联动齐套、计划调整、异常中心和补料记录 |

## 5. 条码与标签整改

| 页面 | 整改要求 |
| --- | --- |
| 生产批次 | “下发批次”改为“生成/发布批次号到工位终端” |
| 产品序列号 | “同步 SN 绑定”改为接收模拟 SN 绑定回执 |
| 物料标签 | 接收 WMS/IQC/线边库或 MES 手工批次标签签收回执 |
| 成品标签 | 关联 FQC 放行、客户标签模板和打印回执 |
| 箱码托盘码 | 接收模拟包装绑定回执，写入包装层级追溯 |
| 标签打印 | 强化模板版本、打印机、首打、补打、作废和审计 |
| 补打申请 | 必须记录原因、权限、新旧标签关系、作废链路和审批 |

## 6. 完工与入库整改

| 页面 | 整改要求 |
| --- | --- |
| 工序完工复核 | 校验数量、工时、用料、质量和 WIP 结转 |
| 完工确认 | 作为入库前校验门，校验 FQC、NCR、SPC、用料和批次谱系 |
| 包装作业 | 包装扫码来自模拟终端回执，绑定 SN、箱码、托盘码 |
| 成品入库 | 有 WMS 接收入库回执；无 WMS 登记 MES 成品库入库记录 |
| 库存冻结 | 有 WMS 下发冻结指令并回执；无 WMS 登记 MES 库存冻结状态 |
| 退料入库 | 接收 PDA/电子秤/WMS 或 MES 手工复核回执 |
| 单据同步 | 业务页看单据状态、差异和补偿结果；设置页只维护补偿规则 |

## 7. 验收标准

- 工位页面不表达为后台直接扫码、投料、报工或交接。
- 物料、仓储页面支持有 WMS 和无 WMS 两种闭环。
- 出入库、投料、退料、包装、标签动作均记录批次、库位或 SN、责任人、时间和追溯引用。
- 所有外部或现场来源动作带“模拟”或“回执”文案。
